## What It Does

Diamond Brain is a self-contained intelligence layer that persists across sessions and agents. Drop it into any Python project for:

- Persistent facts with confidence, time decay, and fuzzy deduplication
- Knowledge graphs and crystallization
- FSRS spaced repetition
- Contradiction detection and confidence propagation
- Legal citation tracking and court document handling
- Encrypted peer-to-peer brain linking (Diamond Link)
- Multi-brain consensus and reputation
- Selective amnesia and live subscriptions
- Visual reports and command memory

Everything is pure Python stdlib + JSON. No external services required.

## Status

Production-ready core with 112+ tests. Actively used as a building block for higher systems.

Precision tools that remember. Built the same way the larger work is built: small, rigorous, compounding pieces that survive.
